# Vim 常用基本操作，与 VsCode 的强强联合

## 基本的模式切换

    vim 操作分四种模式：普通模式(命令模式)、输入模式、视图模式、底线命令模式,普通模式用得最多的模式，而其它三种模式之间的切换则以普通模式为跳板。

1. 普通模式->输入模式：
   - a/A: 字符后/行尾插入
   - o/O: 下/上另起一行插入
   - i/I: 字符前/行首插入
   - s/S: 删除单个字符并进入插入模式/删除整行并进入插入模式
   - c: 配合文本对象进行替换, 文本对象可以理解为一个单词（cw, ciw, caw, 这里 w 就是用来控制单词选择的），一个句子。
   - C: 删除当前行光标所在位置往后的字符

- ctrl + h 删除上一个字符

- 命令模式下：
- 光标移动:
- hjkl 上下左右移动，nh 向左移动 n 个字符
- w,b,e 非空白符分隔的单词
- W,B,E 空白符分隔的单词
- f,F 查找,分号进入下一匹配，逗号返回上一匹配
- t,T 是查找到 f,F 的前一个字符
- 0+w 快速移动到第一行字符
- \$行尾,0 行首,^句首
- ()句子间移动，{}段落间移动
- H,M,L 快速跳转到当前视图开头，中间，结尾
- zz 把屏幕置为中间
- gg/G 快速跳转到最开始和最末尾
- ctrl+o 返回上一次位置
- gi 快速跳转到最后一次编辑的地方
-
- 删除:
- dd 删除整行
- D 向左删除
- x 删除字符
- X 向左删除
- n+d,n+x 删除 n 行或者 n 个字符
- iw: inner word 光标在单词里面任何位置操作单词， diw, di'。
- aw: around word 它会选中当前单词及当前单词之后的空格。 daw,da'。

- 修改:
- r,R 替换并进入插入模式
- c 配合文本对象进行修改,cw, C,ciw,caw 并进入插入模式
- s,S 替换并进入插入模式
- ~大小写切换
-
- 查找:
- /或者?进行向前或反向搜索，n/N 向前或向后匹配，\*/#进行当前单词的向前或向后匹配
- easymotion:
- <leader><leader> t/T <char> 能够向前/向后快速到达可见范围内的某字符(<char>)的位置
- <leader><leader> w/b 向前/向后到达可见范围内任何单词的开头
- <leader><leader> e/ge 向前/向后到达可见范围内任何单词的结尾
- <leader><leader> k/j 向前/向后到达可见范围内任何行的行首
- f,F 查找,分号进入下一匹配，逗号返回上一匹配
-
- 复制粘贴：
- yy,nyy,yw,ye,yb 复制一行，n 行,向后复制一个单词，向前复制一个单词，"9y 复制到 9 号寄存器中
- p 粘贴，"9p 从 9 号寄存器中粘贴
- 寄存器：剪切版'+'，当前文件名'%',上次插入文本'.'
-
- 特殊操作
- .重复上一步操作
- u 类似于 ctrl + z
- <c-r>撤销 u 操作
- <c-d>向下
- <c-u>向上
-
- gd:跳转到定义
- gh:查看光标所在处错误信息
- af: 视图模式下扩展当前块级视图
-
- 注释
- gcc 快速注释当前行, gc2j 注释以下三行, VISUAL 模式 gc 可快速注释选中行
- gC VISUAL 模式使用块级注释
-
-

- @{register}执行宏

- 底部命令模式下：
- :set 查询可用参数
- :set nu 行号
- :set hls 高亮
- :set incsearch 增亮搜索
- :syntax on 语法高亮
- :help ( 查找'('相关帮助信息
- :set clipboard=unnamed 系统剪切版为当前寄存器
- :reg 查看寄存器
- :colorscheme <c-d> {theme} 切换查看当前主题
